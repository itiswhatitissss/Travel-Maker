# 개요

우리의 여행 플래너는 친구와 함께 멋진 여행을 계획하고 조율하는 데 도움을 드립니다. 친구와 함께 파티를 구성하여 플랜을 생성해 일정과 만남장소를 조율하고 참석여부를 확인 할 수 있습니다.  파티가 유지된다면 지나간 플랜도 계속 확인 할 수 있으니, 갤러리 기능과 함께 아름다운 추억을 만들어보세요!
